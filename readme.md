# Hut8 Website

Projeto frontend desenvolvido em React com TypeScript para o site da Hut8, empresa júnior da Computação UFPel.

## 🚀 Tecnologias utilizadas

- React
- TypeScript
- Vite
- CSS Modules

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/Brendler17/hut8.git
```

Acesse a pasta do projeto:

```bash
cd hut8
```

Instale as dependências:
```bash
npm install
```

## ▶️ Executando o projeto

Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

Após iniciar, o terminal exibirá uma URL semelhante a:

```bash
http://localhost:5173
```

Abra essa URL no navegador.
